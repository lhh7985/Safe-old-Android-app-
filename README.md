# Safe-old-Android-app
## 1. 낙상 감지 및 자동 신고 앱

 - 아두이노에서 블루투스로 신호를 받아 알람을 울림
 - 사용자가 알람을 끄지 않을 시 현재 GSP의 주소 정보를 통해 자동 신고


## 2. 프로젝트 정보

    Android Studio, java, googleMap, bluetooth를 사용
    그 외에 아두이노, 블루투스 모듈, 6축 가속도 센서(각속도, 가속도)가 필요합니다.
